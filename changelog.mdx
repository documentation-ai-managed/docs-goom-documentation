## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added Chicago 18th edition citation style with full note-bibliography support.
- Introduced advanced paraphrasing engine using transformer models for natural rephrasing.

## Improvements

- Boosted grammar checker accuracy by 25% with contextual error detection.
- Optimized readability scores to align with Flesch-Kincaid and Gunning Fog indices.

## Bug Fixes

- Fixed MLA in-text citation formatting for sources with multiple authors.
- Resolved export issues in PDF and DOCX formats.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Bulk citation generator: Process up to 50 sources at once.
- Integrated readability assessment with one-click score and suggestions.

## Improvements

- Faster API response times for high-volume requests, now `<200ms` average.

## Bug Fixes

- Corrected APA 7th edition DOI handling for preprints.
- Patched paraphrasing tool to preserve technical terms in STEM content.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core citation generation in APA 7th, MLA 9th, and Chicago 17th styles.
- Basic paraphrasing and grammar checking tools.

## Breaking Changes

- Updated authentication to use `{API_KEY}` header instead of query params.

## Initial Setup

Use the new auth format:

<CodeGroup tabs="JavaScript,Python">
```javascript
const response = await fetch('https://api.citationgenerator.ai/v1/cite', {
  headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` }
});
```

```python
import requests
response = requests.post('https://api.citationgenerator.ai/v1/cite',
                        headers={'Authorization': f'Bearer {YOUR_API_KEY}'})
```
</CodeGroup>

</Update>
